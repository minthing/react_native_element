### react native 기초

##### mac OS 설치하는 법

* expo cli 설치
```
npm install --global expo-cli
```

* watchman 설치
```
brew update
brew install watchman
```

##### make new app
```
expo init weather
cd weather
npm start
```

### rules of react native
* 리엑트 네이티브에서는 `<div>`를 대신해 `View`를 사용함
* Text는 Text에... 재는 재로 돌아가라...
